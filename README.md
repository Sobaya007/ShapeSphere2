# Shape Sphere

## 対応プラットフォーム
Windowsのみ

## ビルドに必要なもの
- D言語用コンパイラ[DMD](https://dlang.org/download.html)
- D言語用ビルドツール[DUB](https://code.dlang.org/download)

## 実行に必要なもの
- [DLL一式](https://drive.trapti.tech/s/kQpXxsvyQTpv6aO)

## ビルド方法
1. このレポジトリをclone
1. `ShapeSphere`に移動
1. `dll`フォルダを作り、ダウンロードしたDLLを全て配置
1. `ShapeSphere`で`dub`と入力
